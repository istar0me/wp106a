# 安裝與查看
1. 安裝 [node.js](https://nodejs.org/en/)
2. 安裝 [npm](https://www.npmjs.com/)
3. 輸入以下指令安裝 [electron](https://electron.atom.io/)

```
npm install -g electron
```

4. 切換到 project 目錄，並輸入
```
electron .
```

* 或透過 RawGit，直接在[瀏覽器中查看](https://goo.gl/N2k6Wv)
